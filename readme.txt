== Description ==

Variation Radio Buttons with Price & Stock for WooCommerce is a powerful plugin that transforms your WooCommerce product variation dropdowns into visually appealing radio buttons. 

With this plugin, customers can instantly see the available variations, including the price and stock levels, without the need for extra clicks. 

This makes the shopping experience smoother and faster, encouraging more purchases and reducing cart abandonment.

See more via [this website](https://shieldclimb.com/free-woocommerce-plugins/variation-radio-buttons/).

== Installation ==

1. Download our plugin from https://shieldclimb.com/free-woocommerce-plugins/variation-radio-buttons/
2. At your WordPress admin, Go to Plugins > Add New Plugin > Upload Plugin > Choose File (Choose the file you downloaded just now) > Click Install
3. Click Activate
